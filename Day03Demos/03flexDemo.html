Learning Flexbox

An html flexbox element typically contains elements that need to be precisely positioned in either horizontal rows or vertical columns. For the purposes of this tutorial, we will only focus on using flexbox to align items in rows. Properties given to the style attribute of the flexbox control how the nested elements are positioned. In the following examples, we'll see what these properties are and what they do.

Note that each one of the items in the flexbox has its height and width defined in the CSS.Try changing the "flex-wrap", "gap", "justify-content", and "align-items" properties above. For each setting, observe what happens when you make the browser very wide and very narrow. This should give you a feeling for what each setting does. Note that these "fixed" items will never grow bigger than the dimensions specified in the CSS. However, if you set the flex-wrap property to "nowrap" and make the browser window narrow enough, the boxes will shrink to remain within the flexbox but never more narrow than their content.

Note that we no longer set the dimensions of each box in the CSS. Instead, we have placed content in each box to dictate the dimensions of the box. You'll find the flex settings work similar to the first example, but now each of the items in the flexbox is only as large as it needs to be to fit its content unless you set "align-items" to "stretch". "Stretch" adjusts the vertical dimension of items whose height is not explicitly specified so that all of these items in the same row have the same height. Note also that this is the default value for "align-items".

â–¸Include a flex setting for items in the flex box

In this example, we give three of the items a flex property value.

HTML:

<div id="flex">
<div id="first">This first box is pretty wide.</div> <div id="second">second</div> <div id="third">
<p>third</p> <p>third</p> <p>third</p> <p>third</p> <p>third</p>
</div> <div id="fourth">
<p>fourth</p> <p>fourth</p>
</div> <div id="fifth">
<p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p>
</div> <div id="sixth">sixth</div>
</div>

CSS:

#first {
background-color: red; padding: 10px;
} #second {
background-color: goldenrod; padding: 10px; flex: 1;
} #third {
background-color: blue; padding: 10px;
} #fourth {
background-color: green; padding: 10px;
} #fifth {
background-color: red; padding: 10px; flex: 1;
} #sixth {
background-color: goldenrod; padding: 10px; flex: 4;
} #flex {
background-color: gray; padding: 10px; display: flex; flex-direction: ; flex-wrap: ; gap: ; justify-content: ; align-items: ;
}

Here we have given three of the boxes their own flex property. The second and fifth box have a flex value of 1 while the sixth box has a flex value of 4. Note that when you widen the browser window, only the boxes to which we have given flex values widens past the width of their content. Furthermore, because we have given the sixth box a flex value that is 4 times the value for the second and fifth boxes, the sixth box accumulates 4 times of the available free space in its row than the second and fifth boxes.

In this next example, we give our flex items a flex basis by including a second parameter for its flex property

HTML:

<div id="flex">
<div id="first">This first box is pretty wide.</div> <div id="second">second</div> <div id="third">
<p>third</p> <p>third</p> <p>third</p> <p>third</p> <p>third</p>
</div> <div id="fourth">
<p>fourth</p> <p>fourth</p>
</div> <div id="fifth">
<p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p> <p>This fifth box is a big box.</p>
</div> <div id="sixth">sixth</div>
</div>

CSS:

#first {
background-color: red; padding: 10px;
} #second {
background-color: goldenrod; padding: 10px; flex: 1 100px;
} #third {
background-color: blue; padding: 10px;
} #fourth {
background-color: green; padding: 10px;
} #fifth {
background-color: red; padding: 10px; flex: 1 content;
} #sixth {
background-color: goldenrod; padding: 10px; flex: 4 200px;
} #flex {
background-color: gray; padding: 10px; display: flex; flex-direction: ; flex-wrap: ; gap: ; justify-content: ; align-items: ;
}

The addition of a basis effectively gives the item a type of minimum width. It's easiest to see the effect of the basis if you set the flexbox to "wrap" and then narrow the browser window. Under these conditions, once any item in a row reaches its basis width, any further reduction in width of the browsing window results in making the last item in the row wrap to the next row. Note that you can make the basis an absolute width (ex. px, em) or you can indicate that the item should use the width of its content to determine its basis by using the word "content" for its basis.
