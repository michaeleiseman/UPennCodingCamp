Learning Float and Clear

The CCS float property is a convenient way to force block or inline elements to the right or left side of a parent container. In practice there are two typical use cases for float. The first use case is for wrapping text around a graphic. The second use case is to stack graphics in an orderly grid that adjusts to the size of the user's window's width.

We demonstrate both use cases below:(Click on the triangles below to see the examples.)

▸Using float to wrap text around an image

You can adjust the float of the colored squares in this example to move them to the right and to the left of the parent container. Note that the text flows around the squares. You can also set the clear property of the text to left or right to make it move below any left or right-floated element respectively.

HTML:

<div class="box red">R</div> <div class="box green">G</div> <div class="box blue">B</div> <p class="text">When an element is floated left, its left edge is moved to the left edge of its parent or a sibling that is also floated left. When an element is floated right, its right edge is moved to the right edge of its parent or a sibling that is also floated right. Other elements are adjacently placed around the floated elements even if they are block elements. This is useful if you want to place text beside an image. If you don't want an element to flow beside a floated image, you can set it's clear property to force it to appear below left and/or right-floated elements.</p>

CSS:

.box {
width:100px; height:100px
} .red {
float: ;
} .green {
float: ;
} .blue {
float: ;
} .text {
clear: ;
}

▸Using float to place boxes in an adjustable grid

You can adjust the float of the colored squares in this example to move them to the right and to the left of the parent container (the body in this case). Try adjusting the width of your web browser to see what happens. Note that the blocks stay neatly packed to the left or to the right when their float property is activated and they wrap to remain within view.

<div class="box red">R</div> <div class="box green">G</div> <div class="box blue">B</div> <div class="box magenta">M</div> <div class="box cyan">C</div> <div class="box yellow">Y</div> <div class="box orange">O</div> <div class="box black">K</div> <div class="box purple">P</div> <div class="box brown">Br</div>
